Cheat Sheet de Etiquetas HTML

Etiquetas básicas

<html>: Define el documento HTML.

<html>
</html>

<head>: Contiene metadatos/información para el documento.

<head>
    <meta charset="UTF-8">
    <title>Título</title>
</head>

<body>: Contiene el contenido visible del documento HTML.

<body>
    <h1>¡Hola Mundo!</h1>
    <p>Este es un párrafo de ejemplo.</p>
</body>

Estructura semántica

<header>: Representa un grupo de ayudas introductorias o de navegación.

<header>
    <h1>Bienvenido</h1>
</header>

<nav>: Define un conjunto de enlaces de navegación.

<nav>
    <a href="#">Inicio</a>
    <a href="#">Acerca de</a>
</nav>

<main>: Representa el contenido principal del documento.

<main>
    <h2>Contenido Principal</h2>
    <p>Este es el contenido principal de la página.</p>
</main>

<aside>: Define contenido aparte del contenido principal.

<aside>
    <p>Contenido relacionado</p>
</aside>

<footer>: Representa el pie de página del documento.

<footer>
    <p>Derechos reservados 2024</p>
</footer>

Párrafos

<p>: Define un párrafo.

<p>Este es un párrafo de ejemplo.</p>

Encabezados

<h1> a <h6>: Definen encabezados de distintos niveles.

<h1>Encabezado 1</h1>
<h2>Encabezado 2</h2>
<h3>Encabezado 3</h3>
<h4>Encabezado 4</h4>
<h5>Encabezado 5</h5>
<h6>Encabezado 6</h6>

Enlaces

<a>: Define un hipervínculo.

<a href="https://www.example.com">Visitar Ejemplo</a>

Listas

<ol>: Define una lista ordenada.

<ol>
    <li>Elemento 1</li
    <li>Elemento 2</li>
</ol>

<ul>: Define una lista desordenada.

<ul>
    <li>Elemento A</li>
    <li>Elemento B</li>
</ul>

<dl>: Define una lista de definiciones.

<dl>
    <dt>Término 1</dt>
    <dd>Definición 1</dd>
    <dt>Término 2</dt>
    <dd>Definición 2</dd>
</dl>

Multimedia

<img>: Define una imagen.

<img src="https://via.placeholder.com/150" alt="Descripción de la imagen">

<audio>: Define contenido de sonido.

<audio controls>
    <source src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3" type="audio/mpeg">
</audio>

<video>: Define contenido de video.

<video controls width="320" height="240">
    <source src="https://www.w3schools.com/html/mov_bbb.mp4" type="video/mp4">
</video>

Tablas

<table>: Define una tabla.

<table border="1">
    <tr>
        <th>Encabezado 1</th>
        <th>Encabezado 2</th>
    </tr>
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>

Formularios

<form>: Define un formulario.

<form action="/submit" method="post">
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
    <input type="submit" value="Enviar">
</form>

<input>: Define un campo de entrada. Variantes:

<input type="text" name="nombre">
    <input type="number" name="edad">
    <input type="tel" name="telefono">
    <input type="email" name="correo">
    <input type="password" name="clave">
    <input type="radio" name="genero" value="masculino">
    <input type="checkbox" name="intereses" value="musica">
    <input type="button" value="Haz clic">
    <input type="color" name="color">
    <input type="date" name="fecha">
    <input type="time" name="hora">
    <input type="datetime-local" name="fechahora">
    <input type="file" name="archivo">
    <input type="hidden" name="secreto" value="valor">

<select>: Define un cuadro combinado.

<select name="opciones">
    <option value="opcion1">Opción 1</option>
    <option value="opcion2">Opción 2</option>
</select>

<textarea>: Define un área de texto.

<textarea name="comentarios">
    Escribe aquí tus comentarios.
</textarea>

<label>: Define una etiqueta para un elemento de formulario.

<label for="nombre">Nombre:</label>
    <input type="text" id="nombre" name="nombre">
